The first step in understanding how to position elements is to understand the fundamentals model of CSS-the box model.

The Box Model

Every element you place on a page creates a box. You cannot alway see the boundries and the borders of the boxes you create, but the boxes are there. A div element creates a box, as do h1,img,span, and td elements. Each box has properties you can set with CSS. I've illustrated some of these properties in Figure 1.

Figure 1 Border, padding, and margin

The padding of a box is the distance between the content of the box and the box's border, and the margin in the distance between a box and any adjacent element. In other words, padding adds space inside the border, while a margin adds space outside the border. In the fllowing CSS, I se both the padding and the margin to a distance of 10 pixels. The border surrounding every element is not visible by default, but by using CSS I can change the color of an element's border, as well as the thickness and line style the browser uses when drawinng the border. What you see in Figure 1 is the result of a style rule telling the browser to draw a solid thin, red line around the box that each div element creates.
